# ⚡ HabitPulse 

HabitPulse is a high-performance, minimalist daily habit logging and analytics dashboard. Engineered with a strict focus on interface ergonomics, zero-overhead client states, and reliable data integrity locks. 

The application provides granular tracking matrices for daily routines, dynamic custom metric ingestion, and visual trailing completion curves.

Live Deployment: https://habitpulse-gamma.vercel.app/

---

## 🛠️ Architecture & Tech Stack

- **Core Runtime Environment:** React + Vite (High-speed HMR compilation pipeline)
- **Styling Architecture:** Tailwind CSS (Fluid utility configurations + structural zinc tokens)
- **Data Visualization Engine:** Recharts (Dynamic SVG area parsing with localized spline gradients)
- **Interface Iconography:** Lucide React
- **State Optimization Layer:** Localized Browser Storage API cache interlocking

---

## ⚙️ Key Structural Engines

### 1. Date-Constrained Calendar Matrix
Renders a fluid 7-column calendar grid that computes daily task completion ratios and dynamically applies intensity-mapped background configurations (Emerald palettes). Implements **strict temporal bounds verification**—disabling interactive mutations or select flags for any timestamps greater than the current real-world execution date.

### 2. Analytical Progress Velocity Module
Maps trailing 30-day performance records into a responsive SVG Area Chart. Automatically groups metrics dynamically to present an accurate completion velocity curve across sessions.

### 3. Responsive Checklist Logging
Manages core baseline logging ("Did I study?", "Did I drink water?", etc.) alongside an expansion array allowing real-time insertion and deletion of custom user parameters with unique thread IDs.
